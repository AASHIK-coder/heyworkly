## Introduction

**heyworkly** is a desktop application that provides a native GUI Agent powered by Vision-Language Models. It allows you to control your computer using natural language instructions.

## Table of Contents

- [Features](#features)
- [Showcase](#showcase)
- [Quick Start](#quick-start)
- [License](#license)

## Features

- Natural language control powered by Vision-Language Model
- Screenshot and visual recognition support
- Precise mouse and keyboard control
- Cross-platform support (Windows/MacOS/Browser)
- Real-time feedback and status display
- Private and secure - fully local processing
- Local and remote computer operators
- Browser automation support
